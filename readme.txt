== Description ==

Simple Menu Delete will add an extra delete button to you menu items. You can now delete menu items with one-click, without having to open the ‘menu edit’ box. 

Related Links:

* http://www.codepress.nl/plugins/simple-menu-delete/

== Installation ==

1. Upload simple-menu-delete to the /wp-content/plugins/ directory
2. Activate Embed Share through the ‘Plugins’ menu in WordPress

A new delete button will be available for each menu item under Appearance > Menus.
